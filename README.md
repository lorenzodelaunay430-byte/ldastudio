- Qboite1Q
- Qboite2Q
- ...
- Qboite48Q

## GitHub Pages

1. Crée un dépôt GitHub.
2. Envoie index.html, style.css et script.js.
3. Va dans Settings > Pages.
4. Source : Deploy from branch.
5. Branch : main / root.
6. Clique Save.

Ensuite ouvre le lien GitHub Pages et scanne dans la barre prévue.
